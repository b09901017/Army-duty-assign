# 排勤務 — 名冊與填公版規則（Project 指示）

## 觸發方式
當使用者說「幫我填公版」「填一下」「填勤務」之類的，並提供「公版」與「簡打」時，就執行下面的填表任務，輸出**整份填好的公版純文字**讓他一鍵複製傳班長。

---

## 班級名冊（261梯，共 8 人）
代號（號碼／學號）= 本名。使用者只要講代號（如「4號」「04」「4」），就是指對應本名：

| 代號 | 本名 |
|---|---|
| 01 / 1 號 | 李愷宸 |
| 02 / 2 號 | 江偉綸 |
| 03 / 3 號 | 陳柏翰 |
| 04 / 4 號 | 鄧旭辰 |
| 05 / 5 號 | 廖翊滕 |
| 06 / 6 號 | 陳俊穎 |
| 07 / 7 號 | 林柏宇 |
| 08 / 8 號 | 林崇浩 |

「261」「261 梯」指的就是我們這班這 8 個人。公版裡凡是寫「261」的地方，就是要我們班填人的位置。

---

## 兩個輸入
1. **公版**：當天完整勤務表，內含別班人名與別梯（258、260）。每天內容可能不同。
2. **簡打**：使用者用「勤務名稱 + 代號」快速指定，例如：
   ```
   上午
   打飯 01
   營部馬路 02 03 06 07 08
   打冰桶 04
   垃圾間 05

   午
   打飯 02

   晚
   打飯 03
   浴廁 八人一起
   ```

---

## 填表規則

### 1) 打掃區（🔷打掃）
- 簡打**有指定**的打掃項目 → 把該行的「261」或「261*N」換成指定本名。
  - 「261」= 1 人；「261*N」= N 人。
  - 名字依**簡打給的順序**用「+」串起來（不留空格），取代掉「261 / 261*N」。
  - 同行的別梯（258、258*n、260）或別班人名 **原樣保留**，只動 261 的部分。
  - 例：`營部馬路：李季恩+261*5+258*n` ＋簡打 `營部馬路 02 03 06 07 08`
    → `營部馬路：李季恩+江偉綸+陳柏翰+陳俊穎+林柏宇+林崇浩+258*n`
- 簡打**沒提到**的打掃項目 → 把該行的「+261」整段拿掉（含前面的「+」），代表我們班沒人去；其他內容不動。
  - 例：`營部走廊：劉育誠+261`（未指定）→ `營部走廊：劉育誠`

### 2) 全班一起 / 保留 261 的情況
- 簡打若寫「八人一起」「全班」「就寫261」 → 該行的「261」**保留不換**，不展開成 8 個名字。
  - 例：`連部浴廁：258+261` → 維持 `258+261`

### 3) 打飯（🔷打飯）
- 打飯固定每天排，分早／午／晚。簡打對應：
  - 上午的「打飯」→ **早打**
  - 午的「打飯」→ **午打**
  - 晚的「打飯」→ **晚打**
- 打飯行通常只有別班人名、沒有「261」字樣。做法是**在原本人名後面接「 + 本名」**（前後留空格），**不是取代**。
  - 例：`早打：郭力瑜` ＋簡打 `打飯 01` → `早打：郭力瑜 + 李愷宸`

### 4) 公差區（🔷公差）與其他沒被簡打提到的區塊
- 一律 **原樣保留，一個字都不要動**（包含「261梯8員」「258+260」「經理公差：6」等）。
- 公差區的數字（如「6」）**不要**當成我們班代號，除非使用者明確指示要填。
- 只有簡打明確指定的項目才動。

---

## 格式慣例
- 打掃區：名字用「+」相接、不留空格（跟公版原本風格一致）。
- 打飯區：用「 + 」（前後空格）接在原本人名後。
- 其餘標點、emoji、換行、附註、日期標題全部照公版原樣輸出。

---

## 輸出與自動檢查
- 直接輸出**整份**填好的公版純文字（含沒動到的部分），方便一鍵複製。
- 輸出後若有以下狀況，主動提醒使用者：
  1. 簡打給的人數與「261*N」的 N **對不上**。
  2. 同一個代號被排到**同時段兩件會衝突**的勤務。
  3. 簡打提到某勤務，但公版**找不到對應行**。
- 對應不明確時 → **先問再填，不要猜**。
